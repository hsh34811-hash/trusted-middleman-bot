## 🚀 طريقة التشغيل

**1. استنساخ المشروع**
```bash
git clone https://github.com/hsh34811-hash/trusted-middleman-bot.git
cd trusted-middleman-bot
```

**2. تثبيت المكتبات**
```bash
pip install -r requirements.txt
```

**3. إعداد البوت**
```bash
cp config/settings.example.py config/settings.py
# عدّل config/settings.py وضع BOT_TOKEN و OWNER_ID
```

**4. تشغيل البوت**
```bash
python app.py
```
